PHP
==========

程式碼風格規範
----------

開發基本上需遵守 [PSR-2](http://www.php-fig.org/psr/psr-2/) 規範。

**注意**：共用元件通常需要支援 PHP 5.5 ，因此建議 PHP 7 的新寫法是不被允許的。

### Array 實字表示

請使用 `[]` 表示，不使用 `array()` 。

```php
// 正確
$arr = [];

// 錯誤
$arr = array();
```

### Array 最後一個元素後需加逗號

範例如下：

```php
// 正確
$arr = [
    'foo',
    'bar',
];

// 錯誤
$arr = [
    'foo',
    'bar'
];
```
