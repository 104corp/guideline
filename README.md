Guideline
==========

本份指南會使用到 [RFC 2119][] 的關鍵字，定義可以參考 RFC 2119 文件說明。以下列出中文對照：

| 原文 | 中文 |
|:--------:|:------:|
| MUST | **必須** |
| MUST NOT | **禁止** |
| SHOULD | **建議** |
| SHOULD NOT | **不建議** |
| MAY | **可以** |

如果有要表達 RFC 2119 文件裡的意思時，將會使用**粗體**表示。
